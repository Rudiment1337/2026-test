м # ✅ Проект №1: Система мониторинга бортовых датчиков автобуса
## 🎯 Цель
- Создать систему сбора, хранения и анализа данных с бортовых датчиков автобуса в реальном времени.

## 📌 Описание
- Система имитирует приём данных от датчиков (температура двигателя, давление в шинах, уровень топлива, скорость).
- Данные обрабатываются, сохраняются, анализируются на аномалии и отображаются в веб-интерфейсе.
- Поддерживает алерты, экспорт, безопасность и интеграцию с внешними системами.

## 🧱 Модели
```
public class SensorData {
    private Long id;
    private Bus bus;
    private SensorType sensorType; 
    private Double value;
    private LocalDateTime timestamp;
    private boolean anomaly;
}

public class Bus {
    private Long id;
    private String model;
    private List<SensorData> lastReadings;
}

public enum SensorType {
    "engine_temp", "tire_pressure", "fuel_level"
}

public class AlertDTO {
    private SensorType sensorType;
    private Action action;
}

public enum Action {
    "ok", "warning", "error"
}
```

## 🌐 API
- `POST /api/sensors` – приём данных от датчика
- `GET /api/sensors/latest?busId=123` – последние показания
- `GET /api/alerts` – список аномалий
- `GET /api/sensors/history?from=...&to=...` – история
- `GET /api/dashboard/stats` – сводка по парку - выгрузка отчета XSLX

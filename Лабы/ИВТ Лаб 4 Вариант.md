# ✅ Проект №4: Система "Умного автобуса" с IoT-интеграцией
## 🎯 Цель
- Создать систему автоматического управления микроклиматом, освещением и развлечениями в салоне.

## 📌 Описание
- Система управляет: кондиционером, светом, музыкой, влажностью.
- Использует данные с датчиков (CO₂, температура) для автоматического режима.
- Поддерживает ручное управление.

## 🧱 Модели
```
public class Bus {
    private Long id;
    private String model;
    private List<Sensor> sensors;
    private String location;
    private Mode mode;
}

public class Sensor {
    private Long id;
    private Bus bus;
    private SensorType type;
    private Double value;
    private LocalDateTime timestamp;
}

public enum SensorType {
    "co2", "humidity", "temp", "light"
}

public enum MusicType {
    "classical", "pop", "hard_rock"
}

public class Mode {
    private Long id;
    private MysicType mysicType;
    private Double targetTemp;
    private Double targeHumidity;
    private Double targetCo2;
}
public enum ModeType {
    "auto", "eco", "comfort"
}

public class SensorTargetDTO{ 
    private SensorType sensorType;
    private Double value;
}
```

## 🌐 API
- PUT /api/mode/{id}/settings – изменить настройки
- GET /api/cabin/status?busId=123 – последнее состояние датчиков
- POST /api/sensors – данные с датчиков

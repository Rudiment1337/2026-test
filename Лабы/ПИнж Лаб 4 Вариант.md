# ✅ Проект №4: Система "Умного дома" с IoT-интеграцией
## 🎯 Цель
- Создать систему автоматического управления микроклиматом, освещением и развлечениями в комнате.

## 📌 Описание
- Система управляет: устройствами дома.
- Использует данные с устройств для автоматического режима.
- Поддерживает ручное управление.

## 🧱 Модели
```
public class Room {
    private Long id;
    private User manager;
    private List<Device> devices;
    private String location;
    private Mode mode;
}

public class Device {
    private Long id;
    private String title;
    private Room room;
    private DeviceType type;
    private Double power;
    private boolean isActive;
}

public enum DeviceType {
    "light", "coffee_machine", "speakers", "kettle", "microwave"
}

public class Mode {
    private Long id;
    private ModeType type;
    private List<Device> devices;
}
public enum ModeType {
    "dinner", "morning", "study"
}

public class DeviceTargetDTO{ 
    private DeviceType deviceType;
    private Double value;
}
```

## 🌐 API
- PUT /api/mode/{id}/settings – изменить настройки
- GET /api/room/status?roomId=123 – последнее состояние устройств
- POST /api/devices – данные с устройств

# ✅ Проект №3: Система учёта времени на проектах
## 🎯 Цель
- Автоматизировать учёт времени, затраченного на учёбу, проекты и дедлайны.
- 
## 📌 Описание
- Студенты отмечают, сколько времени потратили на задачи
## 🧱 Модели
```
public class TimeEntry {
    private Long id;
    private Student student;
    private TaskType type;
    private String description;
    private LocalDateTime start;
    private LocalDateTime end;
    private boolean isBillable; // учётное время
}

public class Student {
    private Long id;
    private String name;
    private String group;
    private List<TimeEntry> recentEntries;
}

public enum TaskType {
    LAB, EXAM_PREP, PROJECT, CODING, REST, LECTURE
}

public class DeadlinePredictionDTO {
    private String subject;
    private LocalDateTime deadline;
    private Double hoursLeft;
    private RiskLevel risk; // LOW, MEDIUM, HIGH
}
```

## 🌐 API
- POST /api/time/start – начать отсчёт времени
- POST /api/time/stop – остановить
- GET /api/time/weekly?studentId=123 – статистика за неделю
- GET /api/predictions/deadline?subject=ИВТ – прогноз успеваемости
- GET /api/reports/export – выгрузка XLSX с графиками

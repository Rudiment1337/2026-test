# ✅ Проект №2: Система мониторинга пожарной безопасности в учебном корпусе
## 🎯 Цель
- Создать систему сбора и анализа данных с пожарных датчиков (дым, температура, CO), автоматического оповещения и формирования отчётов.

## 📌 Описание
- Система получает данные от датчиков, установленных в аудиториях, коридорах, серверных.
## 🧱 Модели
```
public class Alert {
    private Long id;
    private Sensor sensor;
    private EventType type;  
    private LocalDateTime timestamp;
    private String description;
    private StatusType status;
    private List<String> photoUrls;
}

public class Sensor {
    private Long id;
    private String model;
    private String location;
    private User assignedTo;

}

public enum EventType {
    "accident", "hard_braking", "button"
}
public enum StatusType {
    "new", "in_progress", "resolved"
}
```

## 🌐 API
- POST /api/incidents – создание инцидента
- GET /api/incidents?status=new – новые инциденты
- PUT /api/incidents/{id}/assign – назначить ответственного
- PUT /api/incidents/{id}/status_change - сменить статус
- POST /api/incidents/{id}/photos – загрузка фото
